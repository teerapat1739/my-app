# This Project work in 3 steps
  1. run mongodb
  2. Front-end
     * cd to project
     * npm install
     * npm start
    run `code` on `localhost:3000`.
  3. Back-end
     * cd to project and you cd to folder back-end
     * npm install
     * npm start
     run `code` on `localhost:8000`.
